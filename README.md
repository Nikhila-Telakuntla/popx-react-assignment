# 🚀 PopX React Assignment

A responsive ReactJS application developed based on the provided Adobe XD design. The project focuses on creating a pixel-perfect mobile user interface with seamless navigation, clean component architecture, responsive design, and dynamic user data handling using modern frontend development practices.

## ✨ Features

* Welcome Screen
* Login Screen
* Create Account Screen
* Account Settings Screen
* Seamless Navigation Between Pages
* Responsive Mobile-Centered Layout
* Dynamic User Data Rendering
* Reusable React Components
* Pixel-Perfect UI Implementation

## 🛠️ Tech Stack

* ReactJS
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router DOM

## 🎯 Assignment Highlights

* Developed a complete ReactJS application based on the provided Adobe XD design.
* Implemented a pixel-perfect and responsive mobile user interface.
* Built dynamic account creation and login workflows.
* Enabled smooth navigation between all application screens.
* Applied component-based architecture for maintainable and reusable code.
* Followed clean coding standards and modern frontend development practices.
* Prepared the project for production deployment using GitHub and Vercel.
